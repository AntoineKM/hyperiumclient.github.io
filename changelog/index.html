Changelog

The official changelog for Hyperium 1.2.2 Downloads Home

Additions

Customizable Server Button

Quickly join any and all servers with this button! Customizable via holding shift and clicking it.

FOV Modifier

Modify the field of view whenever sprinting, aiming with a bow, have slowness, or have speed!

Support for Optifine 1.8.9 L4

Stay up to date with the latest Optifine version!

ToggleChat Improvements

Remove /boop messages from players using ToggleChat!

Fixes

Nick Hider Improvements

Patched a possible crash when using Nick Hider.

Limbo Detection

Removed the user from the limbo state if they left the server.

Performance

Patched up unclosed streams and connections.
Patched up memory leaks from server icons and tree generation.
Improved Font Renderer performance.
Improved nearby item performance.
Improved startup performance.

Miscellaneous

Other Changes

Client brand is now set to 'hyperium'.
Head Scale max was changed from 2.5 to 3.
Hyperium now logs how long it took to start.
'FPS Limiter Amount' was renamed to 'FPS Limit'.
'Toggle Leather' keybind was changed to NONE by default.

Languages

German and Irish have been updated.
